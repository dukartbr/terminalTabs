## Terminal Tabs

### Riff while you code

To install use:

```
npm install -g github:dukartbr/terminalTabs
```

To use in CLI:

```
terminalTabs
```

Using the following schema, terminalTabs will place your guitar tab in the terminal

```
{
	"name": "Song Title",
	notes: [
		{
			"beat": 0,
			"string": 3,
			"fret": 5
		}
	]
}
```
